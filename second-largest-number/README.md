# Second Largest Number in a List

## Problem
Write a function that returns the second largest distinct number in a list.

## Example

Input:
[1,9,3,8,4,6,1,3,2]

Output:
8

## Explanation
- Largest number is 9
- Second largest distinct number is 8

## Approach
Traverse the list once while keeping track of:
- largest number
- second largest number

Time Complexity: O(n)  
Space Complexity: O(1)
