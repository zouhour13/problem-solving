# Searching Algorithms

## 📌 Overview
This project implements three fundamental searching techniques in Python:

- Linear Search
- Binary Search
- HashMap (Dictionary) Search

These algorithms are commonly used to find elements in a collection and are essential for understanding problem-solving and data structures.

---

## 🚀 Algorithms Included

### 1. Linear Search
Searches for an element by checking each item one by one.

- Works on: Unsorted arrays  
- Time Complexity: O(n)

---

### 2. Binary Search
Searches by repeatedly dividing a sorted array in half.

- Works on: Sorted arrays only  
- Time Complexity: O(log n)

---

### 3. HashMap Search
Uses a dictionary (key-value mapping) for direct access.

- Works on: Key-value data  
- Time Complexity: O(1) average

---

## 📚 What I Learned

- Difference between O(n), O(log n), and O(1)
- How searching works in arrays and dictionaries
- Importance of sorted vs unsorted data
- How HashMaps provide fast lookup using keys
- Basic algorithmic thinking and problem solving

---

## 🧠 Key Takeaways

- Linear Search = simple but slow
- Binary Search = efficient but needs sorted data
- HashMap Search = fastest for direct lookup
